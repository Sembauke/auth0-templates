# Hosted Pages and Templates for Auth0

> Templates used on Auth0 for <https://freeCodeCamp.org>

## Contributing

1. Browse & navigate to the tenant

```console
cd <tenant>
```

2. Install dependencies

```console
npm ci
npm start
```

3. Visit <http://localhost:8800> to see the pages
